# 🍽️ Zaika - A Social Food Recipe App

**Zaika** is a unique food recipe platform where food lovers can share and explore delicious recipes from across the world. With Zaika, users can **add**, **update**, and **delete** their own recipes, **browse recipes by others**, and even **contact the recipe owner** — a unique feature that fosters community and connection among home chefs and foodies.

---

## 🌟 Features

- ✅ Create, update, and delete personal recipes  
- 📖 Browse a collection of public recipes by other users  
- 💬 **Contact the recipe owner** directly through the app  
- 🔒 User authentication and secure access  
- 🖼️ Image support for each recipe  
- 📱 Responsive frontend interface  

---

## 🚀 Getting Started

Follow the steps below to run **Zaika** locally on your machine:

### 🛢️ 1. Start MongoDB

Make sure MongoDB is installed and running locally.

### 📦 2. Start the Backend Server

```bash
cd backend
yarn install
yarn dev
```
---
### 💻 3. Start the Frontend App
In a new terminal window:
```bash
cd frontend
npm install
npm run dev
```
This will launch the frontend of the app at http://localhost:3000.
